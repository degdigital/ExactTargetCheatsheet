Exact Target HTML, CSS, & Ampscript Cheatsheet

HTML

syntax: <tag attribute="value">...</tag>

Tags
<a href="#">...</a> Link
<img src="#" /> Image
<span>...</span> Used to style a
group of text inline
<table>...</table> Table
<tr>...</tr> Table Row
<td>...</td> Table Cell
<strong>...</strong> Defines Strong Text
Attributes
style="..." Specifies an inline style
align="..." Horizontal alignment of cell
Possible values: left, right, center
valign="..." Vertical alignment of cell
Possible values: top, middle, bottom
alt="..." Specifies Alternate Text for an Image
alias="..." Specifies Exact Target Tracking Term
cellpadding="0" Spacing within cells
cellspacing="0" Spacing between cells
class="..." Specifies a classname
Unusable Tags (cause rendering issues)
<p>...</p>
<ul>...</ul>
Unusable Attributes (cause rendering issues)
colspan
rowspan
Deprecated Tags (should be phased out)
<font> Fonts should be declared with CSS on the cell or span level
<b> Bold text should be declared using the font-weight: bold; CSS property or using the <strong> tag
<i> Italic text should be declared using the font-style: italic; CSS property
<center> Text should be centered using the text-align CSS property or using the align HTML attribute
<u> Underlined text should be styled using the text-decoration: underline; CSS property
<strike> Striked text should be styled using the text-decoration: line-through; CSS property
Depricated Attributes (should be phased out)
bgcolor="..." Specify background color using the background or background-color CSS properties
size="" Font size should be declared using the font-size CSS property
